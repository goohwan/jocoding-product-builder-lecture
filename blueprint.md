# Lotto Number Generator

## Overview

This project is a simple web application that generates random lottery numbers. It's built with HTML, CSS, and JavaScript, following modern web development best practices.

## Features

*   Generates 6 unique random numbers between 1 and 45.
*   Displays the generated numbers in a visually appealing way.
*   A button to trigger the number generation.
*   Modern and responsive design.
