# Lotto Number Generator

## Overview

This project is a simple web application that generates random lottery numbers. It's built with HTML, CSS, and JavaScript, following modern web development best practices.

## Features

*   **Dark/Light Mode:** Toggle between light and dark themes for better user experience.
*   **Multiple Sets:** Generates 5 sets of unique random numbers (6 numbers each) at once.
*   **Responsive Design:** Displays the generated numbers in a visually appealing and responsive layout.
*   **Animations:** Smooth animations for number generation and UI interactions.

## Version History

*   **v1.0:** Initial release with single set generation.
*   **v1.1:** Added Dark/Light mode and 5-set generation.
